## ⚠️ Methodology and caveats

This selection is centered on aviation-related domain names rather than a single naming style or extension. The sample set shows broad extension variety, including names like air.lease, flight.social, terminal.place, and tower.im. That makes comparison more important than theme alone. For founders, the best options are usually the names that sound credible, are easy to say, and fit the extension naturally. For investors, discipline starts with ask price, renewal risk, and whether the keyword-extension pairing has plausible buyer demand. A median ask of $1,464 gives a useful reference point, but the stronger names in this set will still depend on clarity, commercial relevance, and extension quality.
